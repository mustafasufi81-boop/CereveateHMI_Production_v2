TURBINE HEALTH KPI MATRIX - IMPLEMENTATION SUMMARY
================================================================================

✅ COMPLETED IMPLEMENTATION (November 24, 2025)

NEW MATRIX STRUCTURE:
--------------------
Instead of grouped tables (Bearings, Shafts, Steam separately),
now shows ONE UNIFIED MATRIX:

ROWS = Universal KPI Metrics (applies to ALL parameters):
  1. Mean Value
  2. Peak Value  
  3. Stability (Std Dev)
  4. Variation Coefficient (CV%)
  5. Health Index (% of design limit)
  6. 95th Percentile

COLUMNS = ALL Selected Parameters (grouped by equipment):
  - 🔩 Bearing Health: HP Front-X, HP Front-Y, HP Rear-X, HP Rear-Y, IP Rear-X, IP Rear-Y
  - ⚙️ Shaft Health: HP Shaft Front/Rear, IP Shaft Rear
  - 🌡️ Steam Path: Main Steam Pressure, Temp, Flow, Reheat Temp
  - ⚡ Load Correlation: Turbine Load MW, Steam Flow, Pressure, Temp
  - ❄️ Cooling Impact: Cooling Water Temp

VISUAL DASHBOARD (3 Interactive Charts):
----------------------------------------
1. Health Index Comparison (Bar Chart)
   - Shows % of design limit for each parameter
   - Color-coded: Green (<60%), Yellow (60-80%), Red (>80%)
   - Threshold lines at 60% and 80%

2. Stability Coefficient CV% (Bar Chart)
   - Shows variation coefficient for each parameter
   - Color-coded: Green (<15%), Yellow (15-30%), Red (>30%)
   - Lower is better (more stable)

3. Mean vs Peak Performance (Grouped Bar Chart)
   - Compares mean and peak values side-by-side
   - Blue bars = Mean, Gold bars = Peak
   - Shows utilization level

CONFIGURATION-DRIVEN:
--------------------
- All parameters from trends-config.json → PivotTableSettings
- Design limits from HealthIndicators → BearingHealthIndex → DesignLimits
- No hardcoded values
- Auto-detection fallback if config missing

KEY FEATURES:
-------------
✅ Matrix format (metrics as rows, ALL parameters as columns)
✅ Color-coded health status (red/yellow/green)
✅ Sticky first column for scrolling
✅ Grouped parameter headers (shows equipment type + icon)
✅ 3 interactive Plotly charts for visual analysis
✅ Health indicator cards (Bearing Health Index, Shaft Stability, Steam Stability)
✅ Responsive design with hover effects

TECHNICAL DETAILS:
------------------
Functions:
- renderTurbineHealthKPIs() - Main matrix builder
- renderVisualDashboard() - Creates 3 chart containers
- renderHealthIndexChart() - Bar chart with threshold lines
- renderStabilityChart() - CV% bar chart
- renderPerformanceChart() - Mean vs Peak grouped bars
- renderDerivedHealthIndicators() - 3 health indicator cards

Data Flow:
1. Get stats from Python API
2. Build parameter groups from config
3. Calculate 6 metrics for each parameter
4. Render matrix table (HTML)
5. Render 3 Plotly charts (after DOM ready)
6. Render health indicator cards

NO HARDCODING:
--------------
- Design limits: from config.PivotTableSettings.HealthIndicators.BearingHealthIndex.DesignLimits
- Parameters: from config.PivotTableSettings.KPIGroups[].Parameters
- Thresholds: from config.PivotTableSettings.HealthIndicators[].GoodThreshold/WarningThreshold
- All calculations dynamic based on available data
