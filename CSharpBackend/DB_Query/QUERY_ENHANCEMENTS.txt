## HOW TO USE:

### Method 1: Quick Query (Recommended)
1. Select tags (Ctrl+Click for multiple)
2. Click a quick time button (e.g., "Last 1 Hour")
3. Click "Execute Query"
4. Done! ✅

### Method 2: Custom Time Range
1. Select tags
2. Set Start Time and End Time manually
3. Click "Execute Query"

### Method 3: Last N Records (All Time)
1. Select tags
2. Click "All Time" button (clears dates)
3. Set Limit to desired number (e.g., 1000)
4. Click "Execute Query"
5. Gets most recent N records regardless of time

## TECHNICAL DETAILS:

### JavaScript Functions:
```javascript
setTimeRange(value, unit)  // Sets start/end times
clearTimeRange()           // Clears dates (All Time mode)
clearResults()             // Resets query results
executeQuery()             // Enhanced with summary
exportData()               // Improved filename
```

### Query Modes:
1. **Time Range + Limit**: Returns up to N records within time range
2. **Time Range Only**: Returns all records in range (up to 100k limit)
3. **All Time + Limit**: Returns most recent N records from entire database
